# 🌆 Fork in the keyboard

This program has an infinite loop when you hold down the W button.

## ❓ FAQ 

#### How to start app?

You need to download the latest release of the program, after that you need to run it as administrator.
##### Right Shift - W Down
##### Delete - W Up
##### Alt + X - Exit App
